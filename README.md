# chart_demo

Demo ứng dụng Flutter minh hoạ 3 thư viện biểu đồ: **fl_chart**, **Syncfusion**, **graphic**.

## Cách chạy

```bash
# 1. Cài dependencies
flutter pub get

# 2. Chạy app
flutter run
```

> Nếu nhận code từ Drive (không có thư mục `build/`), chỉ cần chạy 2 lệnh trên là đủ.
